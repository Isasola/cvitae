Verificar localmente\npnpm build\n\n# 2. Revisar errores\npnpm check  # TypeScript\n\n# 3. Limpiar y reinstalar\nrm -rf node_modules pnpm-lock.yaml\npnpm install\n\n# 4. Intentar build nuevamente\npnpm build\n\n# 5. Si funciona localmente, push a GitHub\ngit add .\ngit commit -m \"fix: build errors\"\ngit push origin main\n```\n\n### Error: \"Deployment timeout\"\n\n**Causa:** Build tarda más de 15 minutos\n\n**Solución:**\n- Optimizar assets (no guardar imágenes grandes en `/public`)\n- Usar CDN para imágenes: `manus-upload-file --webdev`\n- Revisar dependencias innecesarias\n\n### Error: \"Cannot find module\"\n\n**Causa:** Dependencia faltante en Netlify\n\n**Solución:**\n```bash\n# Asegurar que pnpm-lock.yaml está en Git\ngit add pnpm-lock.yaml\ngit commit -m \"fix: add pnpm-lock.yaml\"\ngit push origin main\n```\n\n### Sitio muestra versión vieja\n\n**Causa:** Cache del navegador\n\n**Solución:**\n- Limpiar cache: Ctrl+Shift+R (Windows/Linux) o Cmd+Shift+R (Mac)\n- Forzar purge en Netlify: **Settings → Deploys → Trigger deploy**\n\n---\n\n## Seguridad en Producción\n\n### Checklist de Seguridad\n\n- [ ] HTTPS habilitado (automático en Netlify)\n- [ ] No
